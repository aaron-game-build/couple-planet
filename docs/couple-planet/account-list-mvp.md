# 账号列表（免重复登录）MVP

## 功能范围

- 登录页展示已保存账号列表。
- 支持点击账号一键切换（token-only）。
- 支持删除单个账号与清空全部账号。
- token 失效时提示重新登录并标记账号状态。

## 存储模型

- `userId`
- `account`
- `nickname`
- `token`
- `lastUsedAt`
- `needsRelogin`

存储方式：`flutter_secure_storage`，单 key JSON 列表。

## 关键交互

1. 手动登录成功后：写入/更新账号列表并按关系态分流。
2. 点击已保存账号后：
   - 设置 token
   - 调 `auth/me` 校验
   - 成功分流到 Chat/Bind
   - 失败标记 `needsRelogin=true` 并提示重新登录

## 验收标准

- 登录过的账号在登录页可见，按最近使用排序。
- 点击有效账号无需密码即可进入业务流。
- 失效账号不会静默失败，会提示并要求重新登录。
- 删除/清空后账号不再可切换。
- 切换账号后聊天发送身份正确，无串号。

## 风险控制

- 不保存密码，仅保存 token。
- 不打印 token 到日志。
- token 失效时主动降级到“需重登”状态。
