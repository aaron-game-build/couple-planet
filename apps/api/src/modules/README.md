# Week1 模块落位映射

## auth

目录：`src/modules/auth`

接口：

- `POST /auth/register`
- `POST /auth/login`
- `GET /auth/me`

## relation

目录：`src/modules/relation`

接口：

- `POST /relations/invite-code`
- `POST /relations/bind`
- `GET /relations/current`

## chat

目录：`src/modules/chat`

接口：

- `GET /chats/current/messages`
- `POST /chats/current/messages`
- `POST /chats/current/messages/read`
